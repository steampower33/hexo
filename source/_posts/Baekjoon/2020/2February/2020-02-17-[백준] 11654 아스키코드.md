## 2. 현재 코드

```c++
#include <iostream>
using namespace std;

int main(){
    char a;
    cin >> a;

    cout << static_cast<int>(a) << endl;
    return 0;
}
```

## 3. 설명

static_cast를 통해서 char형으로 입력받았던 값을 int형으로 변환한다.
