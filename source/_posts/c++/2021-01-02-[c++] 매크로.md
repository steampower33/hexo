# 간결한 코드 작성을 위한 c++ 매크로 사용

```c++
bool TrueOrFalse(const vector<int>& array) {
    for(int i = 0; i < array.size(); ++i) 
        for(int j = 0; j < i; ++j)
            if(array[i] == array[j])
                return true;
    return false;
}
```

```c++
#define FOR(i,n) for(int i = 0; i < (n); ++i)
bool TrueOrFalse(const vector<int>& array) {
    FOR(i, array.size())
        FOR(j,i)
            if(array[i] == array[j])
                return true;
    return false;
}

```
